I had stopped programming for a while as I've been working as a SRE engineer in my new company (since 2020/08).

Recently, I started touching code again and weird things happened. It took me almost a day to troubleshoot a problem on running Kafka in Docker which I could tackle easily and quickly before. I forgot basic knowledge of Kafka and hardly run Kafka commands to operate a cluster.

Actually, it's logical. The thing is I didn't realize I had stopped programming for so long. I kept thinking that these are not my stuffs right now. Oh, dear! Programming may be not your stuff right now but you know that you don't want to be bad at programming. Ever.
