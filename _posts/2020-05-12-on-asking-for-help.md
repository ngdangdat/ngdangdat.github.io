# Background
* I had been asking for help from other team desperately day after day in two weeks.
* My company's development team is using SCRUM and has 3 week sprint.

# Realization
It should have been easier for me if I knew how to do it properly.

# Lession learned
* If it’s urgent and predictable -> Jira task should be asked and prepared in sprint planning meeting.
* If it’s urgent and not predictable -> should ask PO in the sprint so that they can understand the priority.

**Note:** It’s better to apply this to small but critical (means if it’s not done, your task will be blocked) tasks like creating MySQL user or granting permission on test environment.
